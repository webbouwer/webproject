HTML5 basis cross browser

Dit is een basis HTML5 Template met CSS en Javascript code. Samen maken deze codes een complete webpagina. Dit template is responsive door CSS en interactief door het gebruik van Javascript.

Javascript is open source en voor iedereen te gebruiken. Omdat javascript al lang bestaat hebben verschillende webdevelopers code bedacht die het gebruik van Javascript makkelijker maakt. Wanneer je een klein beetje van Javascript begrijpt en kent, kun je de JQueryof andere javascript Libraries gaan gebruiken.
